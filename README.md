### Importing and building from Command Line

* `cd` into the directory you would like to host the movies app

~~~bash
$> cd ~/MY_PARENT_FOLDER
~~~

* Run the project

~~~bash
$> appc ti clean && appc run -p ios -T simulator
~~~
